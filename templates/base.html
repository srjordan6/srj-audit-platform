{% comment %} Site-wide base layout template. Every page on aiauditforcompanies.com extends this template unless it has a special reason not to (e.g., HTMX partial fragments that get swapped in — they emit fragments, not full pages). CDN choices (locked at Sprint C PR 3) ------------------------------------- Bootstrap 5.3 — jsdelivr CDN, integrity-hashed. Sourced from https://www.jsdelivr.com/package/npm/bootstrap HTMX 2.0 — unpkg CDN. HTMX 2 is the current stable version. SortableJS — jsdelivr CDN. Needed by the RANK question partial. CDN-based delivery is the right choice for Sprint C because we don't yet have a build pipeline (no Vite, no npm, no static-asset processing beyond Django's collectstatic). If we later add a build step for Tailwind or brand-specific CSS, we can switch to bundled assets without touching the template contract — pages just extend this file. Django-htmx integration ----------------------- The tag provides the CSRF value in a predictable location for HTMX. Combined with django-htmx middleware's hx-headers config, HTMX requests automatically include the token.
